üîí Tuple

IMMUTABLE ¬∑ ORDERED ¬∑ DUPLICATES OK
MethodWhat it doesReturns
.count(x)Number of times x appearsint
.index(x)Index of first occurrence of xint

Tuples only have 2 methods because they're immutable. Use built-in functions like len(), sorted(), min(), max(), sum() on them.

Example
coords = (10, 20, 10, 30) print(coords.count(10)) # 2 print(coords.index(20)) # 1 # Tuple unpacking x, y, *rest = coords # x=10, y=20, rest=[10,30] # Named tuple (more useful tuple) from collections import namedtuple Point = namedtuple("Point", ["x", "y"]) p = Point(3, 4) print(p.x, p.y) # 3 4
